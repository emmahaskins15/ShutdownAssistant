# ShutdownAssistant

Schedule a time for your computer to shut down.

Compatible with Windows 10, probably 7 or older as well.

Requires .NET Core Runtime

# Features
* Schedule shutdown
* Abort scheduled shutdown
* Portable

# Usage
Extract .ZIP and run ShutdownAssistant.exe
